# Architecture charts

These charts give maintainers a quick visual map of how the plugin, docs, and validation workflows fit together.

## Available charts

- `Plugin architecture`
- `Rule lifecycle and sync flow`

## When to consult these charts

Use them when you are:

- adding a new rule and need to remember all required integration points
- debugging docs or preset drift
- onboarding a new maintainer to the repository layout
