# Security Policy

## Reporting a Vulnerability

If you discover a vulnerability in this repository, report it privately through one of these channels:

- GitHub Security Advisory draft: <https://github.com/Nick2bad4u/eslint-plugin-vite/security/advisories/new>
- Maintainer email: <20943337+Nick2bad4u@users.noreply.github.com>

Do **not** open public issues for unpatched vulnerabilities.

## Supported Versions

Only the latest published release is actively supported for security fixes.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ |
| Older   | ❌ |

## User guidance

- Keep `eslint-plugin-vite`, ESLint, TypeScript, Vite, and Vitest updated.
- Run linting in CI on trusted code.
- Review rule messages and config changes before applying them broadly.

## Credits

Responsible disclosure is appreciated. Reporters can be credited in release notes on request.
